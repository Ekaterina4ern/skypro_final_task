Git

  • git init - Инициализация нового репозитория
  • git clone - Клонирование удалённого репозитория
  • git add - Добавление файлов в индекс
  • git commit - Синхронизация изменений
  • git push - Отправка изменений на сервер Github
  • git pull - Получение изменений с сервера Github
  • git branch - Работа с ветками
  • git checkout branch name of new branch - Переключение с ветки на ветку
